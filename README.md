# Dsembler
DNA assembly designer

## INSTALL
git clone git@github.com:sblabkribb/dsembler.git

## Pre-requisit
docker, docker-compose, mongoDB

## start
  - GO to the dsembler directory
  - docker-compose up -d
  - docker exec -it dsembler /bin/bash
  - python dsembler.py


## stop
docker-compose down
